ProCaptchera

ProCaptchera is an open-source captcha software that includes 2 types of the security captcha, the 2 types are randomly generated to each visitor submitting your form, some visitors automatically get the $captchatype = "math"; and some other visitors get $captchatype = "imagecaptcha"; If you're wondering what's those codes, download ProCaptchera now and discover the functions of this framework by yourself :)

Features

ProCaptchera have got a lot of features, from them:

  • Easy Integration.
  • Open-source licensed.
  • Active Development.
  • High security levels (2 security types).
  • Easy to Read Read-Me guide.
  • Image Captcha is customizable.
  • Easy-to-read (ETR) and Easy-to-edit (ETE) framework.
